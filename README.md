# Sistema de Modulación AM en Python

Este proyecto implementa un sistema de modulación en amplitud (AM)
utilizando Python.

## Incluye:
- Señal de mensaje
- Señal modulada
- Ruido
- Atenuación
- Distorsión
- Análisis FFT

## Resultados:
Se analizó el impacto del ruido, la atenuación y la distorsión
en la señal modulada.
